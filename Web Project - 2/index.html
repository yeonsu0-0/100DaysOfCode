HTML & CSS Basics Summary

HTML Summary

HTML (HyperText Markup Language) is the heart of every webpage. It defines the structure of the webpage and annotates the content to tell the browser what to display and provide extra meaning.

HTML (HyperText Markup Language) is the heart of every webpage.

It defines the structure of the webpage and annotates the content to tell the browser what to display and provide extra meaning.

Without HTML, text would always be "just text"

there would be no semantic difference between titles, normal text, subtitles etc.

Even though browsers by default add styles for certain HTML elements (e.g. elements are bold and bigger by default), you don't use HTML for styling purposes!

CSS Summary

CSS (Cascading Style Sheets) is a language that allows you to define style definitions for your HTML document.

These definitions will be picked up by the browser

control how the content will be displayed

Where HTML defines the structure and meaning of your content

CSS is used to then present the content exactly how you want to present it.

It adds no meaning or annotation - it's really just about how things look.
